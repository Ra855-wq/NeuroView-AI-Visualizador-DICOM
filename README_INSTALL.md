
# Instruções de Instalação - NeuroView AI

Este projeto utiliza módulos modernos de JavaScript e IA. Para rodar corretamente, siga os passos abaixo:

## Linux / macOS
1. Abra o terminal na pasta do projeto.
2. Dê permissão de execução ao script:
   ```bash
   chmod +x setup.sh
   ```
3. Execute o instalador:
   ```bash
   ./setup.sh
   ```

## Windows
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Abra o Prompt de Comando (CMD) ou PowerShell na pasta.
3. Execute:
   ```cmd
   npm install && npm start
   ```

## Requisitos
- Node.js v18 ou superior.
- Conexão com a internet para carregar bibliotecas via CDN e usar a IA.
